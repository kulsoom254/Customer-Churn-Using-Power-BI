Project Overview:
 This project analyzes customer churn data using Power BI to identify patterns and key factors influencing customer retention. The goal is to provide actionable insights through interactive dashboards and well-defined KPIs.

Technical Details:
 Data Processing-

  *Cleaned and transformed data using Power Query Editor
  *Removed duplicates and handled missing values
  *Standardized data types and column names
  *Created calculated columns for analysis

*Demographics
*Contract duration
*Service subscriptions
*Provided insights to support data-driven retention strategies
